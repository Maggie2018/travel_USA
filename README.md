Note: 新奥尔良花园区以精美的老建筑而闻名，这里除了有众多希腊复兴式和哥特式的建筑之外，还有很多有新奥尔良特色的景观，附近还有很多博物馆可以随心逛逛，感受宁静、历史和艺术。花园区的特别之处，并不是其高雅的花园，而是独一无二的
